todowork

A todo framework

Install

 
npm i todowork

or

yarn add todowork

Usage

 
import { TodoList } from 'todowork';

// Create a new list
const list = new TodoList();

// Add some stuff
list.items.push({
    name: 'Make a to-do app',
    complete: false
});

// Clear list
list.clear();
